Bubble Sort

Algorithm

Description: Bubble Sort repeatedly steps through the array, compares adjacent items, and swaps them if they are in the wrong order. The pass through the array is repeated until the array is sorted.

BUBBLE-SORT(A, n):
1. for j = n down to 1
2.   for i = 1 to j-1 
3.       if A[i] > A[i+1] 
4.           swap(A[i], A[i+1])
Time Complexity

Best Case: Ω(n) (when already sorted)
Average Case: Θ(n²)
Worst Case: O(n²)

Space Complexity

Auxiliary Space: O(1)
Total Space: O(n) (array itself + few variables)

Selection Sort

Algorithm

Description: Selection Sort divides the array into a sorted and unsorted part. It repeatedly selects the minimum element from the unsorted part and moves it into the sorted part.

SELECTION-SORT(A, n): 
1. for i = 0 to n-2
2.     min = i
3.     for j = i+1 to n-1
4.         if A[j] < A[min]
5.             min = j
6.     swap(A[i], A[min])
Time Complexity

Best: Ω(n²)
Average: Θ(n²)
Worst: O(n²)

Space Complexity

Auxiliary Space: O(1)
Total Space: O(n)

Insertion Sort

Algorithm

Description: Insertion Sort works the way you might sort playing cards in your hands. Elements are picked one by one from the unsorted section and inserted into their correct position in the sorted section.

INSERTION-SORT(A, n): 
1. for i = 1 to n-1
2.     key = A[i]
3.     j = i-1
4.     while j >= 0 and A[j] > key
5.         A[j+1] = A[j]
6.         j = j - 1
7.     A[j+1] = key
Time Complexity

Best: Ω(n) (already sorted)
Average: Θ(n²)
Worst: O(n²)

Space Complexity

Auxiliary Space: O(1)
Total Space: O(n)

Quick Sort

Algorithm

Description: Quick Sort uses a divide-and-conquer strategy. The array is partitioned into two smaller subarrays using a pivot, and then those subarrays are sorted recursively.

QUICKSORT(A, low, high) 
1. if low < high
2.     p = PARTITION(A, low, high)
3.     QUICKSORT(A, low, p-1)
4.     QUICKSORT(A, p+1, high)
PARTITION(A, low, high) 
1. pivot = A[high]
2. i = low - 1
3. for j = low to high-1
4.     if A[j] <= pivot
5.         i++
6.         swap(A[i], A[j])
7. swap(A[i+1], A[high])
8. return i+1
Time Complexity

Best: Ω(n log n)
Average: Θ(n log n)
Worst: O(n²)

Space Complexity

Average Case: O(log n) extra
Worst Case: O(n) extra

Merge Sort

Algorithm

Description: Merge Sort is another divide-and-conquer sorting algorithm. The array is recursively divided into halves, then merged together in sorted order.

MERGE-SORT(A, p, r) 
1. if p < r
2.     q = floor((p+r)/2)
3.     MERGE-SORT(A, p, q)
4.     MERGE-SORT(A, q+1, r)
5.     MERGE(A, p, q, r)
MERGE(A, p, q, r) 
1. n1 = q - p + 1
2. n2 = r - q
3. create arrays L[1...n1], R[1...n2]
4. copy into L and R
5. merge them back into A
Time Complexity

Best: Ω(n log n)
Average: Θ(n log n)
Worst: O(n log n)

Space Complexity

Auxiliary Space: O(n) (temporary arrays)
